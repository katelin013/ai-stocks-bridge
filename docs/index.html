Connect your browser to local AI CLI tools.
Analyze stocks with Gemini, Claude, and Codex -- all running on your machine.

Security

  • Localhost only -- no external connections accepted
  • Zero network requests -- Bridge never phones home
  • Zero dependencies -- only Node.js built-in modules
  • Fully open source -- under 100 lines of code
  • CORS restricted -- only your AI Stocks app can connect
